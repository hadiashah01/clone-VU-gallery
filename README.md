## Objective

The objective of this project is to clone the design of the Virtual University Photo Gallery webpage using HTML and CSS only.
This project focuses on copying the layout and visual appearance, not the functionality.


## Concepts Covered

The following HTML and CSS concepts were practiced in this project:

- HTML page structure

- CSS display (block, inline-block)

- margin and padding

- border

- box-sizing

- text-align

- color and background-color
